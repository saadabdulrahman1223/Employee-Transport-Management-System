# Employee Transport Management System (ETMS)

## Overview
Python-based desktop application to manage employee transport operations, including employee-driver assignments, vehicle management, and routes. Features simulated notifications and professional GUI.

## Features
- Add, update, and manage employees, drivers, vehicles, and routes
- Assign employees to drivers using dropdown menus
- Simulated notifications for assignments
- Reset and clear database data buttons
- Professional GUI using Python Tkinter
- Database integration with MySQL

## Technologies Used
Python, Tkinter, MySQL

## How to Run
1. Install required packages: `pip install mysql-connector-python`
2. Configure your MySQL database
3. Run `main.py` to start the application
